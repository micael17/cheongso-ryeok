# 청소기 비교 분석 작성 워크플로우

2~4개 제품을 비교 분석하는 글을 작성합니다.

## Step 1: 비교 대상 확인

사용자에게 다음을 확인하세요:

1. **비교할 제품들** (2~4개)
   - 제품명, 브랜드
   - 같은 카테고리인지 확인 (무선청소기끼리, 로봇청소기끼리)

2. **비교 목적**
   - 가격대 비교? (예: 50만원대 무선청소기)
   - 브랜드 비교? (예: 다이슨 vs LG)
   - 특정 기능 비교? (예: 물걸레 기능 비교)

## Step 2: 비교 항목 설정

제품 유형에 따른 핵심 비교 항목:

### 무선청소기
- 흡입력 (AW/W)
- 배터리 시간
- 무게
- 먼지통 용량
- 특수 기능 (레이저, 디스플레이 등)
- 가격

### 로봇청소기
- 흡입력 (Pa)
- 물걸레 기능
- 장애물 회피
- 자동 비움/세척
- 앱 기능
- 가격

### 물걸레청소기
- 물걸레 방식 (회전/진동)
- 물탱크 용량
- 자동 세척 기능
- 배터리 시간
- 가격

## Step 3: 자료 수집

각 제품별로 다음 자료 필요:
- 공식 스펙
- YouTube 리뷰 요약
- 사용자 후기 요약

자료가 부족하면 WebSearch/WebFetch로 보완하세요.

  - "{제품A 전체명}"
  - "{제품B 전체명}"

winner: "{추천 제품명}"

## Step 5: winner 필드 작성 가이드

- 확실한 승자가 있으면: `"다이슨 V15"`
- 상황별 추천이 다르면: `"다이슨 V15 (성능) / LG A9S (가성비)"`
- 무승부면: winner 필드 생략
